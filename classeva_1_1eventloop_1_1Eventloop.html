def eva.eventloop.Eventloop.sort_queue (   self)

Sort queue according to EVA_QUEUE_ORDER.

This function guarantees that:

  • RPC requests are put first in the queue in FIFO order
  • If using the FIFO order, messages are put in chronological order
  • If using the LIFO order, messages are put in reverse chronological order
  • If using the ADAPTIVE order, messages are put in chronological order, but messages with a more recent reference time are put first in the queue.
